# Microsoft Stock Price Analysis 📈

An Excel-based analysis of Microsoft (MSFT) stock data — exploring price trends, trading volume patterns, and monthly performance over time.

## What this project covers

- Daily Open, High, Low, Close prices and Volume
- Month-on-month price trend analysis
- Identifying high-volume trading days
- Visual charts for price movement over time

## Key findings

- Stock showed consistent upward trend through late 2020 into early 2021
- Notable volume spikes observed around major market events
- Highest single-day volume recorded in January 2021
